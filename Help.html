How to operate?


Guidlines:
1) Use "e" to indicate "Ɛ" (epsilon).
2) Use enter or space to separate each Regular Grammer.
3) "->" is used to indicate "arrow".
4) The First RG is always the first state
5) After the arrow, each transition and their state is divided by "|". e.g., S->0A|0B
6) To enter an epsilon transition, just type the next state. e.g., S->A|B.
7) Press "e" for final state. e.g., S->1A|e.
8) To import file press the import button. Only text file and regular grammar files are supported.
9) Clear button will clear all the fields and views
10)The states named are merged for DFA and Minimized DFA due to clarfication for new states origin.

-End of user manual-
